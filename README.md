# ЛР5 Flask

В работе доработан проект ЛР4: добавлена авторизация по ролям через декоратор `check_rights` с ограничением доступа к действиям и скрытием недоступных кнопок в интерфейсе, реализован журнал посещений `visit_logs` с автозаполнением через `before_request`, а также вынесенный в Blueprint модуль статистических отчетов (журнал с пагинацией, отчет по страницам, отчет по пользователям и экспорт обоих отчетов в CSV). Тестовые учетные записи: администратор `admin / Admin123!`, пользователь `user01 / User123!`.
